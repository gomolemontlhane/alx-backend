## Getting Started

### Prerequisites

This project requires the following dependencies:
- **Programming Language:** C
- **Package Manager:** Npm, Autotools, Cmake

### Installation

Build alx-backend from the source and install dependencies:

1. **Clone the repository:**
   ```
   git clone https://github.com/gomolemontlhane/alx-backend
   ```
2. **Navigate to the project directory:**
   ```
   cd alx-backend
   ```
3. **Install the dependencies:**

   **Using npm:**
   ```
   npm install
   ```

   **Using cmake:**
   ```
   cmake . && make
   ```

### Usage

Run the project with:

**Using npm:**
```
npm start
```

**Using cmake:**
```
./alx-backend
```

### Testing

Alx-backend uses the {**test_framework**} test framework. Run the test suite with:

**Using npm:**
```
npm test
```
**Using cmake:**
```
ctest
```
